# Orikyx Multitool

A powerful multitool for networking, SSH, and more!

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OrikyxOnIG/Orikyx-Multitool.git

Install Go (if not already installed):
```
https://go.dev/doc/install
```

Build the tool:
```bash
cd Orikyx-Multitool
go build
```

Run the tool
```bash
./orikyx-multitool
